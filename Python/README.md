# Zahlenratespiel

## Ziel des Spiels
Schreibe ein kleines Programm, das sich eine zufällige Zahl zwischen 1 und 100 ausdenkt.
Der Spieler soll diese Zahl erraten.
- Nach jedem Tipp soll das Programm sagen, ob die Zahl zu hoch oder zu niedrig war.
- Der Spieler hat eine begrenzte Anzahl Versuche (z. B. 10).
- Das Spiel endet, wenn der Spieler die Zahl errät oder alle Versuche aufgebraucht sind.
- Am Ende wird dem Spieler angezeigt, ob er gewonnen oder verloren hat und – optional – die richtige Zahl.

### Extra-Aufgabe (optional)
Miss die benötigte Zeit, bis der Spieler die Zahl errät oder das Spiel endet, und zeige sie am Ende an.
