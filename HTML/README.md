# Button Beispiel

## Ziel
Erstelle eine kleine Webseite, auf der ein Button die Überschrift auf der Seite ändert, wenn er geklickt wird.
Die Webseite soll folgende Funktionen haben:
- Eine Überschrift (z. B. „Noch nicht gedrückt“).
- Einen Button mit Text wie „Klick mich“.
- Wenn der Button gedrückt wird, ändert sich der Text der Überschrift zu „Button wurde gedrückt!“.
- Die Seite soll gestaltet sein, damit sie gut aussieht:
    - Ein Hintergrund (Farbverlauf oder Farbe)
    - Der Button sollte sichtbar, klickbar und mit Hover- oder Klick-Effekten versehen sein
    - Die Überschrift sollte gut lesbar sein
